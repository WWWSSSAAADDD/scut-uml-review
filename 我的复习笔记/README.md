## 🧭 学习方法（约定）

- **节奏**：周末突击 + 工作日 1h 复习（艾宾浩斯曲线友好）
- **语言**：讲解中文；定义/清单/答题模板用英文（卷子是英文）
- **画图**：Mermaid 主力 + 每周 30 分钟手画肌肉记忆（考前 1 周每日 1 张 POS）
- **教学风格**：苏格拉底式 + 反向考试（GRASP / 顺序图 / 案例分析），简单概念直讲
- **主案例**：Larman POS 系统（贯穿 13 个笔记），每节末加"举一反三"做迁移
