## 答题英文模板（考试时直接写）

```
The Unified Process (UP) consists of four phases:
- Inception:    define scope, identify key requirements, business case
- Elaboration:  build core architecture, address risks, refine requirements
- Construction: implement remaining elements iteratively
- Transition:   beta release, user feedback, deployment
```
