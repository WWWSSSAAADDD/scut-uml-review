## 学习目标

### 状态图（State Diagram）
- [ ] 状态、转移、事件、动作、守卫条件
- [ ] 进入/退出动作、do 活动
- [ ] 复合状态与并发状态
- [ ] 历史状态（H, H*）

### 活动图（Activity Diagram）
- [ ] 动作、控制流、决策、合并、并发（fork/join）
- [ ] 泳道（swimlane）
- [ ] 与流程图的区别

### 包图（Package Diagram）
- [ ] 包、依赖、嵌套
- [ ] 应用：分层架构（UI / Domain / Service / Data）
